Linear Regression Demo

1. Linear Regression Formulas:

If we want to approximate \(y\) with a linear regression line \( ax+b \), then the formulas for \(a\) and \(b\) are given as

\[ a = \frac{n \sum x_i y_i - \sum x_i \sum y_i}{n \sum x_i^2 - (\sum x_i)^2}, \]

\[ b = \frac{\sum y_i - a \sum x_i}{n}. \]
